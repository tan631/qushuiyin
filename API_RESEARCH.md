# 免费接口调研报告

## 1. YouTube（推荐优先级：⭐⭐⭐⭐⭐）

### 方案 1：YouTube Data API v3
- **免费额度**：每天 10,000 配额单位
- **功能**：获取视频信息、字幕、缩略图
- **限制**：不能直接下载视频文件
- **申请**：Google Cloud Console 免费开通
- **成本**：完全免费（配额内）

### 方案 2：yt-dlp（推荐）
- **类型**：开源命令行工具
- **功能**：下载视频 + 提取字幕 + 元数据
- **部署**：需要服务器端运行
- **成本**：完全免费
- **稳定性**：⭐⭐⭐⭐⭐（社区活跃维护）
- **GitHub**：https://github.com/yt-dlp/yt-dlp

**测试代码**：
```bash
# 安装
pip install yt-dlp

# 提取视频信息
yt-dlp --dump-json "https://www.youtube.com/watch?v=VIDEO_ID"

# 下载视频
yt-dlp -f "best[ext=mp4]" "URL"
```

## 2. TikTok/抖音（推荐优先级：⭐⭐⭐⭐）

### 方案 1：TikTok API（官方）
- **免费额度**：需申请开发者账号
- **限制**：审核严格，个人开发者难申请
- **不推荐原因**：申请门槛高

### 方案 2：开源解析库（推荐）
- **工具**：tiktok-scraper (Node.js)
- **功能**：无水印视频下载 + 文案提取
- **成本**：完全免费
- **风险**：可能被反爬，需要定期更新
- **稳定性**：⭐⭐⭐⭐

**测试代码**：
```javascript
const TikTokScraper = require('tiktok-scraper');

const videoMeta = await TikTokScraper.getVideoMeta('https://www.tiktok.com/@user/video/123');
// 返回：视频 URL、描述、作者、点赞数等
```

## 3. Twitter/X（推荐优先级：⭐⭐⭐⭐）

### 方案 1：Twitter API v2 Free Tier
- **免费额度**：每月 1,500 条推文读取 + 50 条推文发布
- **功能**：获取推文文本、媒体 URL
- **限制**：需申请开发者账号（审核较快）
- **成本**：完全免费（配额内）
- **稳定性**：⭐⭐⭐⭐⭐

**申请流程**：
1. 访问 https://developer.twitter.com
2. 申请开发者账号
3. 创建 App 获取 API Key

**测试代码**：
```bash
curl -X GET "https://api.twitter.com/2/tweets/1234567890?expansions=attachments.media_keys&media.fields=url" \
  -H "Authorization: Bearer YOUR_BEARER_TOKEN"
```

## 4. 小红书（推荐优先级：⭐⭐⭐）

### 现状
- **无官方 API**

### 方案 1：第三方 API 服务
- **RapidAPI 上的小红书 API**
- **免费层**：每月 100 次请求
- **成本**：免费层后按次收费
- **稳定性**：⭐⭐⭐

### 方案 2：自建爬虫
- **技术**：Puppeteer + 代理池
- **风险**：⚠️ 反爬严格，需要处理验证码、IP 封禁
- **合规性**：⚠️ 需注意小红书服务条款
- **稳定性**：⭐⭐

**建议**：暂缓开发，或使用第三方付费 API

## 5. Instagram（推荐优先级：⭐⭐⭐）

### 方案 1：Instagram Basic Display API
- **免费额度**：完全免费，但有速率限制
- **功能**：获取用户媒体（图片/视频）
- **限制**：
  - 需要 Facebook 开发者账号
  - 只能访问授权用户自己的内容
  - 不能访问公开帖子
- **不适合**：提取任意链接内容

### 方案 2：开源工具 Instaloader
- **类型**：Python 库
- **功能**：下载帖子、故事、视频
- **成本**：完全免费
- **风险**：⚠️ Instagram 反爬严格，容易被限制
- **稳定性**：⭐⭐

**建议**：暂缓开发，或使用付费 API
